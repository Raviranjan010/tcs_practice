# 🤝 Interview Prep (Post-NQT Selection Rounds)

Once you clear the TCS NQT written cutoffs, you will be invited to a face-to-face or virtual panel interview. This panel typically consists of three interviewers covering **Technical (TR)**, **Managerial (MR)**, and **Human Resources (HR)** competencies.

---

## 🗂️ Interview Modules

| Document | Focus Area | Key Preparation Topics |
| :--- | :--- | :--- |
| 📍 [Technical Interview](Technical_Interview.md) | Technical Round (TR) | OOPS, DBMS, SQL, Projects, Core DSA |
| 📍 [MR Round](MR_Round.md) | Managerial Round (MR) | Situation questions, Agile, Tech Trends |
| 📍 [HR Questions](HR_Questions.md) | HR Round | Communication, Relocation, Shift work |
| 📍 [Do & Don'ts](DO_DONT.md) | Conduct Rules | Dress code, body language, speaking rules |
