# 🤝 Contributing to TCS NQT Prep Hub

Thank you for helping us improve this repository! Contributions from students and software engineers make this resource better for everyone.

## How to Contribute
1. Fork this repository.
2. Create a new branch: `git checkout -b feature/topic-addition`.
3. Add your solved C++ code or revision notes.
4. Ensure your files comply with line count constraints and include step-by-step math derivations.
5. Open a Pull Request.
