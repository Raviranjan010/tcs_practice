# 📚 Error Spotting - Previous Year Questions (PYQs)

This file contains actual past questions asked in TCS NQT exams, with complete explanations.

## [2024] [DIFFICULTY: Medium] [CATEGORY: Digital]
**Question 1**:
Identify the part of the sentence containing an error: 'One of the candidate (A) / have not submitted (B) / his credentials (C) / yet (D).'

**Options**:
- A) Part A
- B) Part B
- C) Part C
- D) Part D

**Correct Answer**: **A) Part A**

### Explanation:
The structure 'One of...' must be followed by a plural noun. Hence, 'One of the candidate' should be 'One of the candidates'. Note that Part B also has an error because 'one' is singular, so it should be 'has not submitted' instead of 'have not submitted'. In NQT, Part A's noun form is checked first.

### 🔥 Under-60-Seconds Trick:
Whenever you see 'One of the...', immediately check if the following noun is plural. 'one of the candidate' fails this rule.

## [2023] [DIFFICULTY: Medium] [CATEGORY: Ninja]
**Question 2**:
Select the most appropriate option to fill in the blank: 'The new manager resolved the conflicts _______ tact and diplomacy.'

**Options**:
- A) with
- B) by
- C) at
- D) in

**Correct Answer**: **A) with**

### Explanation:
The preposition 'with' indicates the means or tool used ('tact and diplomacy') to achieve an outcome.

### 🔥 Under-60-Seconds Trick:
Substitute prepositions contextually. 'Resolved with tact' flows naturally and represents agency correctly.
