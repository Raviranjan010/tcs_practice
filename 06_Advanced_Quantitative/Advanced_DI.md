# 📊 Advanced Data Interpretation

Advanced Data Interpretation tests your skill in extracting, joining, and projecting values from multiple nested charts (e.g. Pie chart + Table) and Radar graphs.

---

## 🕸️ Radar Charts Guide

Radar charts represent multi-variable data plotted on 2D axes starting from the same central point.

* **Axes**: Each axis represents a specific category or attribute (e.g. Sales, Profit, Headcount).
* **Reading Values**: Follow the concentric grid lines outwards from the center. Values increase as you move away from the center.

---

## ⚠️ Common Pitfalls

* **Double Counting**: In mixed charts, failing to verify if percentages overlap. Always read footnotes to see if categories are mutually exclusive.
* **Scale Differences**: Mixing up scales (e.g. one axis in millions, another in thousands). Convert all metrics to a common denominator before performing ratios.
