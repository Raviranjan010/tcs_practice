## [2024] [DIFFICULTY: Easy] [CATEGORY: Ninja]
**Question 1**:
Which flowchart shape is used for 'Print Sum'?

**Options**:
- A) Rectangle
- B) Parallelogram
- C) Oval
- D) Diamond

**Correct Answer**: **B) Parallelogram**

### Explanation:
Printing output is an I/O operation. All I/O operations are shown inside a parallelogram.

### 🔥 Under-60-Seconds Trick:
Output = I/O = Parallelogram. Direct mapping.

## [2023] [DIFFICULTY: Medium] [CATEGORY: Ninja]
**Question 2**:
What is the output of the following pseudocode?
Integer x = 4
Integer y = x >> 1
print y

**Options**:
- A) 8
- B) 2
- C) 1
- D) 4

**Correct Answer**: **B) 2**

### Explanation:
Right shift by 1 is equivalent to division by 2. 4 / 2 = 2.

### 🔥 Under-60-Seconds Trick:
$x \gg 1$ divides the number by 2. $4/2 = 2$. Instant solve.
