## 💡 Prep Strategy for TCS NQT Programming Logic
1. **Understand Bitwise Operations**: TCS NQT pseudocode is notorious for testing bitwise XOR (`^`), AND (`&`), OR (`|`), and shifts (`<<`, `>>`). Memorize their truth tables.
2. **Watch Loop Increments**: Pay extremely close attention to loops that do not increment by 1 (e.g. `i = i + 2`) or have nested increments inside the body.
3. **Trace Variable Scope**: In output questions, differentiate between global and local scopes and trace variable changes step-by-step.
