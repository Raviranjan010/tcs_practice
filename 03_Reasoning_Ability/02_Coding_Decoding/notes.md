# 📝 Coding Decoding - Comprehensive Reasoning Notes

> **Concept Definition**:
> Coding Decoding evaluates pattern recognition in word transformations. It covers letter shifting, reverse alphabet positions, skip coding, number substitutions, and symbol substitutions. Mastering alphabet ranks (A=1 to Z=26) and reverse ranks (Z=1 to A=26) is essential.

## 📊 Logic & Analytical Rules

| Rule Index | Analytical Principle | Diagram / Application |
| :--- | :--- | :--- |
| **Rule 1** | A=1, B=2, ..., Z=26 rankings should be memorized using EJOTY (5, 10, 15, 20, 25). | Analytical Layout |
| **Rule 2** | Reverse positions: rank of letter from right = 27 - rank from left. (e.g. Z = 27 - 26 = 1). | Analytical Layout |
| **Rule 3** | Identify the pattern by comparing corresponding letters of word and code: check for +1, -1, +2, -2 shifts, or alternating shifts. | Analytical Layout |
| **Rule 4** | Reverse word coding: check if the coded word is simply the original word written backwards (e.g., DELHI -> IHLED). | Analytical Layout |
| **Rule 5** | Number substitution: letters are replaced by their rank sum or individual ranks. | Analytical Layout |

## ✍️ Worked Examples

### Example 1:
If ROME is coded as TQOG, how is PARIS coded?

A) A) RCTKU
B) B) QBSJT
C) C) SBTLV
D) D) RDVLU

**Answer**: `A`

**Step-by-Step Logic**:
1. Analyze letters of ROME to TQOG:
   - R -> (+2) -> T
   - O -> (+2) -> Q
   - M -> (+2) -> O
   - E -> (+2) -> G
2. The pattern is +2 shift for every letter.
3. Apply this to PARIS:
   - P -> (+2) -> R
   - A -> (+2) -> C
   - R -> (+2) -> T
   - I -> (+2) -> K
   - S -> (+2) -> U
4. Coded word is RCTKU. Option A.

### Example 2:
If FRIEND is coded as 60, what is the code for ENEMY?

A) A) 61
B) B) 62
C) C) 60
D) D) 65

**Answer**: `B`

**Step-by-Step Logic**:
1. Check letter ranks for FRIEND:
   F=6, R=18, I=9, E=5, N=14, D=4.
2. Sum of ranks = 6 + 18 + 9 + 5 + 14 + 4 = 60.
3. Apply to ENEMY:
   E=5, N=14, E=5, M=13, Y=25.
4. Sum = 5 + 14 + 5 + 13 + 25 = 62.
5. Coded value is 62. Option B.
