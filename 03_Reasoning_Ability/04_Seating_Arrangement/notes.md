# 📝 Seating Arrangement - Comprehensive Reasoning Notes

> **Concept Definition**:
> Seating Arrangement covers linear rows, circular layouts, and double row configurations. It requires placing people based on relative clues (immediate left, second to right, opposite).

## 📊 Logic & Analytical Rules

| Rule Index | Analytical Principle | Diagram / Application |
| :--- | :--- | :--- |
| **Rule 1** | In circular seating facing inward: Clockwise = Left, Counter-clockwise = Right. | Analytical Layout |
| **Rule 2** | In circular seating facing outward: Clockwise = Right, Counter-clockwise = Left. | Analytical Layout |
| **Rule 3** | Linear rows facing North: Left is left, Right is right. Facing South: directions invert. | Analytical Layout |
| **Rule 4** | Start placing elements with fixed, absolute clues first, rather than conditional ones. | Analytical Layout |

### Example 1:
Eight persons A, B, C, D, E, F, G, H are sitting in a linear row facing North. A sits third from the left end. B is not adjacent to C. D sits to the immediate right of E. H sits at one of the extreme ends. G sits third to the right of F. Determine the seating arrangement.

A) A) H-E-D-A-F-B-G-C
B) B) E-D-A-F-B-G-C-H
C) C) A-B-C-D-E-F-G-H
D) D) H-A-B-C-D-E-F-G

**Answer**: `A`

6. Verification: B is not adjacent to C (B is 6, C is 8 - verified). D is immediate right of E (E=2, D=3 - verified). H is at end (H=1 - verified). G is third to right of F (F=5, G=8? Wait, 5+3=8.
