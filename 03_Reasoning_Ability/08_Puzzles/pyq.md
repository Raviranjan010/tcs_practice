## [2024] [DIFFICULTY: Hard] [CATEGORY: Prime]
**Question 1**:
Three boxes A, B, and C contain balls. One box has red, one has blue, and one has green. A label is pasted on each: 'Red', 'Blue', 'Green'. All labels are incorrectly pasted. If Box A's label is 'Red' and it actually contains green balls, what is inside Box B if its label is 'Blue'?

**Options**:
- A) Red
- B) Blue
- C) Green
- D) Cannot be determined

**Correct Answer**: **A) Red**

### Explanation:
A has label 'Red' (actual is Green). All labels are wrong.
Labels are: A=Red, B=Blue, C=Green.
Box contents: A has Green. Since all labels are wrong, B (labeled Blue) cannot contain Blue. B cannot contain Green (already in A). Thus, B must contain Red.
C (labeled Green) must contain Blue.

### 🔥 Under-60-Seconds Trick:
All labels wrong means B (labeled Blue) cannot be Blue. Since A has Green, B must have the only remaining color: Red. Option A. Solved in 10 seconds.

## [2023] [DIFFICULTY: Medium] [CATEGORY: Ninja]
**Question 2**:
In a certain code, 'DOG' is coded as '26' and 'CAT' is coded as '24'. What is the code for 'PIG'?

**Options**:
- A) 32
- B) 42
- C) 30
- D) 36

**Correct Answer**: **A) 32**

### Explanation:
Sum of position values: P=16, I=9, G=7. Sum = 16+9+7 = 32.

### 🔥 Under-60-Seconds Trick:
Recall alphabet mappings: P(16) + I(9) + G(7) = 32. Direct sum.
