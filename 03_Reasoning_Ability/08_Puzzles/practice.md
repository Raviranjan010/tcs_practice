## 🟡 Intermediate Level (Questions 11 - 20)

**Q11.** A, B, C are red, blue, green. A is not green. B is blue. What color is A?
<details>
<summary>Click to view Answer & Explanation</summary>

* **Answer**: `Red`
* **Method**: Grid elimination.
* **Step-by-Step Solution**:
  Since B is blue, A and C cannot be blue. A is not green, so A must be red. C must be green.
* **Target Time**: 20 seconds
</details>

## 🔴 Advanced Level (Questions 21 - 30)

**Q21.** A, B, C work in Dev, QA, DevOps. DevOps person has a blue car. A has a red car. C is from Dev. Who works in DevOps?
<details>
<summary>Click to view Answer & Explanation</summary>

* **Answer**: `B`
* **Method**: Double variable grid elimination.
* **Step-by-Step Solution**:
  C is from Dev. A has a red car, so A is not the DevOps person (who has a blue car). Hence B must be in DevOps (with a blue car). A must be QA.
* **Target Time**: 40 seconds
</details>
