# 📝 Data Interpretation - Comprehensive Reasoning Notes

> **Concept Definition**:
> Data Interpretation evaluates quantitative analysis of charts, including bar graphs, pie charts, line graphs, and tables. Questions test percentage growth, ratios, and averages.

## 📊 Logic & Analytical Rules

| Rule Index | Analytical Principle | Diagram / Application |
| :--- | :--- | :--- |
| **Rule 1** | Perform estimations rather than exact divisions to save time. | Analytical Layout |
| **Rule 2** | Read axes labels and units (thousands, millions) carefully. | Analytical Layout |
| **Rule 3** | Percentage growth = (New Value - Old Value) / Old Value * 100. | Analytical Layout |

## ✍️ Worked Examples

### Example 1:
The following table shows sales (in thousands) of 4 companies A, B, C, D over 3 years:

| Company | 2020 | 2021 | 2022 |
| :--- | :---: | :---: | :---: |
| A | 100 | 120 | 150 |
| B | 80 | 90 | 110 |
| C | 150 | 160 | 180 |
| D | 200 | 220 | 250 |

Find the percentage growth of company A from 2020 to 2022.

A) A) 20%
B) B) 50%
C) C) 25%
D) D) 30%

**Answer**: `B`

**Step-by-Step Logic**:
1. Sales of A in 2020 = 100. Sales in 2022 = 150.
2. Increase = 150 - 100 = 50.
3. Percentage growth = (50 / 100) * 100 = 50%.
4. Option B is correct.
