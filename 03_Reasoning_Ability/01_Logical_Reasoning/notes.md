# 📝 Logical Reasoning - Comprehensive Reasoning Notes

> **Concept Definition**:
> Logical Reasoning tests statement analysis, assumptions, course of action, cause and effect, and strong/weak arguments. It requires drawing objective deductions strictly based on statements provided, avoiding external facts or assumptions.

## 📊 Logic & Analytical Rules

| Rule Index | Analytical Principle | Diagram / Application |
| :--- | :--- | :--- |
| **Rule 1** | Assumptions are implicit parts of statements; an assumption is valid if it must be true for the statement to hold. | Analytical Layout |
| **Rule 2** | Conclusions must follow 100% logically from statements. If there is a 1% chance it doesn't follow, it is invalid. | Analytical Layout |
| **Rule 3** | In cause-and-effect, determine which event occurred first or triggered the other. If both are independent, identify common causes. | Analytical Layout |
| **Rule 4** | Course of action must be a practical, logical, and immediate solution that reduces the problem without creating new ones. | Analytical Layout |
| **Rule 5** | Strong arguments are directly related, logical, and backed by facts or major policies, while weak arguments are superficial, emotional, or make simple analogies. | Analytical Layout |

## ✍️ Worked Examples

### Example 1:
Statements:
All dogs are animals.
Some animals are wild.

Conclusions:
I. Some dogs are wild.
II. All animals are dogs.

A) A) Only I follows
B) B) Only II follows
C) C) Both follow
D) D) Neither follows

**Answer**: `D`

**Step-by-Step Logic**:
1. Let's draw the Venn diagrams:
   - Circle 'Dogs' is completely inside circle 'Animals' (All dogs are animals).
   - Circle 'Wild' overlaps with circle 'Animals' (Some animals are wild).
2. Conclusion I: 'Some dogs are wild'. There is no definite overlap between circle 'Dogs' and circle 'Wild'. It is a possibility, but not a definite conclusion. Thus, I does not follow.
3. Conclusion II: 'All animals are dogs'. The circle 'Animals' is larger and contains 'Dogs'; thus, all dogs are animals, but all animals are not dogs. Thus, II does not follow.
4. Neither conclusion follows. Option D is correct.
