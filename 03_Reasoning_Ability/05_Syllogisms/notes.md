# 📝 Syllogisms - Concept & Short Tricks

> **Concept Definition**:
> Syllogisms deal with statements that are accepted as true, and conclusions that must be verified as logically following from them. Use Venn Diagrams to find truth values.

## 📊 Key Logic / Concept Table

| Reasoning Concept | Analytical Rule |
| :--- | :--- |
| All A are B | Draw circle A entirely inside circle B. |
| Some A are B | Draw circle A overlapping circle B. |
| No A is B | Draw circle A and B completely separate. |

## ✍️ Worked Examples (Step-by-Step)

### Example 1:
**Question**: Statements: All pens are pencils. All pencils are erasers.
Conclusions: I) All pens are erasers. II) Some pencils are pens.
*Question*: Which conclusions follow?

**Solution**:
Step 1: Draw Venn diagram. Circle 'Pens' inside 'Pencils'. Circle 'Pencils' inside 'Erasers'.
Step 2: Check Conclusion I: 'Pens' is entirely inside 'Erasers', so it follows.
Step 3: Check Conclusion II: 'Pencils' circle overlaps 'Pens' circle, so it follows.
Step 4: Both conclusions follow.

### Example 2:
**Question**: Statements: Some keys are locks. Some locks are doors.
Conclusions: I) Some keys are doors. II) No key is a door.
*Question*: Which conclusions follow?

**Solution**:
Step 1: Draw primary case. Keys overlaps Locks, Locks overlaps Doors. (Keys and Doors separate).
Step 2: Draw alternate case. Keys overlaps Locks, Locks overlaps Doors, and Keys overlaps Doors.
Step 3: Conclusion I is true in case 2, false in case 1. Conclusion II is true in case 1, false in case 2.
Step 4: Since they form a complementary pair (Some + No), the answer is Either I or II follows.

### Example 3:
**Question**: Statements: No dog is cat. All cats are animals.
Conclusions: I) Some animals are cats. II) No dog is animal.
*Question*: Check validity.

**Solution**:
Step 1: Draw Cats inside Animals. Dog is separate from Cats.
Step 2: Check Conclusion I: Since Cats is inside Animals, there must be some overlap. True.
Step 3: Check Conclusion II: Animals can overlap Dog while keeping Cats separate. So not necessarily true.
Step 4: Only conclusion I follows.

## ⚠️ Common Mistakes

❌ **Pitfall**: Assuming 'Some A are not B' is the same as 'Some A are B'. They are not equivalent. In logic, 'Some' means 'at least one', and we must draw all possibilities to be absolutely certain.

## 💡 Clue / Shortcut Trick

🔥 **Shortcut**: Recognize the 'Either-Or' conditions: 1) Same subjects/predicates. 2) One conclusion positive, one negative. 3) Both conclusions are false individually.
