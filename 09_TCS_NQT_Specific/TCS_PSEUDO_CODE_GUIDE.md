# 💻 TCS Proprietary Pseudocode Syntax Guide

TCS NQT uses a unique pseudocode notation in its programming logic section. This guide provides the complete syntax reference.

---

## 📝 Syntax Reference Table

| Structure | TCS Notation | Equivalent C / Python |
| :--- | :--- | :--- |
| **Variable Declaration** | `Integer x, y` | `int x, y;` |
| **Assignment** | `x = 10` | `x = 10;` |
| **Conditionals** | `if (condition)... else... end if` | `if (condition) {} else {}` |
| **Loops** | `while (condition)... end while` | `while (condition) {}` |
| **Bitwise AND** | `x & y` | `x & y` |
| **Bitwise OR** | `x | y` | `x | y` |
| **Bitwise XOR** | `x ^ y` | `x ^ y` |
| **Right / Left Shift** | `x >> 1`, `x << 1` | `x >> 1`, `x << 1` |

## ✍️ 20 Practice Pseudocode Questions

### Q1. What is the output of the following pseudocode?
```text
Integer a, b, c
a = 5
b = 2
a = a ^ b
b = a ^ b
print a + b
```
* **Options**:
  - A) 7
  - B) 12
  - C) 9
  - D) 7
* <details><summary>🔍 View Answer & Explanation</summary>
  
  **Correct Answer**: A) 7  
  **Explanation**: This is the standard XOR swap algorithm. The values of `a` and `b` are swapped. Thus, the sum `a + b` remains identical: `5 + 2 = 7`.  
  **Trick**: XOR swaps do not change the sum of the variables. Sum is always constant.  
  **Time to Solve**: 15 seconds.
  </details>
