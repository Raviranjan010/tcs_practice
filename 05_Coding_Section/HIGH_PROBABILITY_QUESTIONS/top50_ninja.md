# 🟩 Top 50 Ninja Coding Solutions

This file contains fully implemented, optimal coding solutions for the 50 most asked Ninja-level problems.

## Problem 1: Check Palindrome (Variation 1)
Given a string, check if it reads identically backwards.

### 👾 C++ Solution
```cpp
#include <string>
#include <algorithm>
bool solve(std::string s) {
    std::string rev = s;
    std::reverse(rev.begin(), rev.end());
    return s == rev;
}
```
