## Problem 1: Two Sum Hash Map (Variation 1)
Find indices of two numbers that sum to target.

### 🐍 Python Solution
```python
def two_sum(arr, target):
    seen = {}
    for idx, x in enumerate(arr):
        if target - x in seen:
            return [seen[target - x], idx]
        seen[x] = idx
    return []
```

### 👾 C++ Solution
```cpp
#include <vector>
#include <unordered_map>
std::vector<int> twoSum(std::vector<int>& arr, int target) {
    std::unordered_map<int, int> seen;
    for(int i = 0; i < arr.size(); ++i) {
        if(seen.count(target - arr[i])) return {seen[target - arr[i]], i};
        seen[arr[i]] = i;
    }
    return {};
}
```

### 🐍 Python Solution
```python
def two_sum(arr, target):
    seen = {}
    for idx, x in enumerate(arr):
        if target - x in seen:
            return [seen[target - x], idx]
        seen[x] = idx
    return []
```

### 👾 C++ Solution
```cpp
#include <vector>
#include <unordered_map>
std::vector<int> twoSum(std::vector<int>& arr, int target) {
    std::unordered_map<int, int> seen;
    for(int i = 0; i < arr.size(); ++i) {
        if(seen.count(target - arr[i])) return {seen[target - arr[i]], i};
        seen[arr[i]] = i;
    }
    return {};
}
```
