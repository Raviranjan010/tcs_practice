# 📝 Recursion & Backtracking Coding Challenges

Attempt these challenges under 30 minutes each. Code files must handle all boundary constraints.

1. **Reverse String / Array**: Given array of numbers, reverse it without auxiliary arrays.
2. **Target Sum**: Find two numbers that sum up to target (Use Hashmap/Two Pointer).
3. **Rotate Array**: Rotate an array by K elements in $O(N)$ time and $O(1)$ space.
