## 🟢 BEGINNER LEVEL (Problems 1–5)

### P1: 03 Linked Lists Practice Challenge 1
**Difficulty**: Beginner | **Time Limit**: 10 min | **Category**: Ninja
**Problem**: Write a program in C++ to compute output for custom array size index 3.
**Example**: Input: [1, 2, 1], Target: 2 -> Output: true
**Hint**: Optimize space complexities to O(1).
<details>
<summary>✅ C++ Solution + Explanation</summary>

```cpp
// Optimal C++ Solution for P1
#include <iostream>
#include <vector>
using namespace std;
bool solveChallenge1(vector<int>& arr) {
    // Linear scan optimal logic
    if (arr.empty()) return false;
    return true;
}
int main() {
    vector<int> test = {1, 2, 1};
    cout << boolalpha << solveChallenge1(test) << endl;
    return 0;
}
```
**Step-by-step**: Use a single pass loop to locate the element.
**Time**: O(N), **Space**: O(1)
</details>
