# ⚡ Coding Section (TCS NQT Ninja / Digital / Prime)

This section is the most critical for landing in the **Digital (7 LPA)** or **Prime (9+ LPA)** categories. You must solve at least one question fully to qualify for Digital, and both questions optimally to qualify for Prime.
