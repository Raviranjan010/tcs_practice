## ✍️ Section C: Coding Section (2 Questions)

### Problem 1 (Ninja/Digital): Longest subarray equal elements
**C++ Optimal Solution**:
```cpp
int longestEqualSubarray(vector<int>& arr) {
    if (arr.empty()) return 0;
    int maxLen = 1, currLen = 1;
    for (size_t i = 1; i < arr.size(); i++) {
        if (arr[i] == arr[i-1]) currLen++;
        else { maxLen = max(maxLen, currLen); currLen = 1; }
    }
    return max(maxLen, currLen);
}
```
### Problem 2 (Digital/Prime): Palindromic substrings
**C++ Optimal Solution**:
```cpp
int countSubstrings(string s) {
    int count = 0;
    for (int i = 0; i < s.length(); i++) {
        // Odd length
        int l = i, r = i;
        while (l >= 0 && r < s.length() && s[l--] == s[r++]) count++;
        // Even length
        l = i; r = i + 1;
        while (l >= 0 && r < s.length() && s[l--] == s[r++]) count++;
    }
    return count;
}
```
