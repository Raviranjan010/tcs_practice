# 📝 Logarithms - Comprehensive Study Notes

> **Concept Definition**:
> Inverse of exponentiation. Includes logarithmic properties, equations, base conversion, and finding characteristics and mantissas.

## 📊 Formula Reference Table

| Concept / Formula | Equation | Description / Usage |
| :--- | :--- | :--- |
| **Product Rule** | `$log(x * y) = log(x) + log(y)$` | Multiplication splits to addition |
| **Quotient Rule** | `$log(x / y) = log(x) - log(y)$` | Division splits to subtraction |
| **Power Rule** | `$log(x^y) = y * log(x)$` | Exponent becomes multiplier |
| **Base Change Formula** | `$log_b(a) = log_c(a) / log_c(b)$` | Used to switch bases |

## ✍️ 5 Worked Examples (Step-by-Step)

### Example 1:
**Question**: Find value of log_2(32).

**Step-by-Step Solution**:
log_2(32) = log_2(2^5) = 5 * log_2(2) = 5.

**Correct Answer**: `5`

### Example 2:
**Question**: If log2 = 0.3010, find log5.

**Step-by-Step Solution**:
log5 = log(10 / 2) = log10 - log2 = 1 - 0.3010 = 0.6990.

**Correct Answer**: `0.6990`

## ⚠️ EXAM TRAPS (TCS Specific)

1. **Writing log(x+y) = log(x) + log(y) which is completely incorrect.**
2. **Forgetting that logs are only defined for positive real bases and arguments.**

## 🔥 60-Second Shortcuts & Speed Tricks

- **Trick 1**: log_a(b) * log_b(a) = 1.

## 🧠 Memory Aid & Mnemonic
**Mnemonic**: log(xy) = logx + logy. log(x/y) = logx - logy. log(x^p) = p logx.

Always remember to solve problems by converting variables to their lowest prime bases (2, 3, 5, 7, 11...). This renders complex divisions into simple exponent arithmetic operations.
